### 🛠 사용 기술

- front-end: `typescript` `next.js` `redux` `redux-saga` `styled-components`
- back-end: `express` `sequelize(mysql)`
- server: `aws ec2`
- apis: `한국관광공사_국문 관광정보 서비스` `kakao maps`

### 🏷️ 구현 기능

- 로그인, 회원가입
- 카테고리별 관광지
- 검색 기능
- 정렬 (인기순, 최신순)
- 댓글 추가, 수정, 삭제
- 카카오맵을 이용한 길찾기
- 반응형, 스켈레톤
- custom hooks (input, toggle)

### 💡성능최적화

- 폰트 ttf에서 woff로 변환
- meta tag를 추가해 SEO 개선
- usecallback을 통한 렌더링 최적화
- next/image로 이미지 최적화
